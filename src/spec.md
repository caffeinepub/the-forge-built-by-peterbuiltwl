# The Forge – Built by PETERBUILTWL

## Overview
A comprehensive portfolio management system that consolidates all AI applications under the "PETERBUILTWL + $DSWENTWORTH" brand into a unified control center with centralized dashboard, multi-app portfolio, donation system, and role-based access management.

## Core Features

- Centralized navigation hub with menu: Dashboard | Apps | Payments | Donations | Implementation Library | About
- Real-time KPI display:
  - Revenue today
  - Active subscriptions
  - Runs today

  - Donations today
- Live activity feed showing chronological log of payments, app runs, and donation transactions
- Dark corporate visual theme with aqua-cyan gradients
- Consistent footer branding: "PETERBUILTWL + $DSWENTWORTH"

### App Portfolio Integration
Complete suite of AI applications, each following the Universal Workflow System (Input → Payment → AI Prompt → Output):

1. **Niche Blog/Newsletter Generator** - Topic, tone, audience inputs; subscription model; generates 500-800 word articles
2. **Resume Optimizer** - Resume upload and job description inputs; one-time payment; optimized resume output
3. **Ad Copy Generator** - Product/service details and platform inputs; per-use payment; targeted ad copy output
4. **Blog Writer** - Topic and style inputs; subscription model; full blog post output
5. **Social Media Scheduler** - Content and platform inputs; subscription model; scheduled post content output
6. **Cover Letter Builder** - Job details and resume inputs; one-time payment; personalized cover letter output
7. **FAQ Generator** - Product/service information inputs; one-time payment; comprehensive FAQ list output
8. **Product Description AI** - Product details and target market inputs; per-use payment; compelling product descriptions output
9. **Email Campaign Writer** - Campaign goals and audience inputs; subscription model; email sequence output
10. **Presentation Builder** - Topic and audience inputs; one-time payment; presentation outline and content output
11. **Podcast Script Maker** - Topic and format inputs; subscription model; structured podcast script output

Each app includes consistent input forms, integrated payment processing, AI-generated content, and output delivery with PDF download and copy-to-clipboard functionality.

### Universal Workflow System
Standardized four-stage workflow across all applications:

1. **Input Stage**: Dynamic forms with text fields, dropdowns, and file uploads based on app requirements
2. **Payment Stage**: Integrated Stripe/PayPal processing with subscription and one-time payment options
3. **AI Prompt Stage**: App-specific AI content generation
4. **Output Stage**: Results display with text output, PDF download, and copy-to-clipboard features

### Donation System ($DSWENTWORTH)

- Donation KPI tracking: Daily, Monthly, and Lifetime totals

### App Wizard
- Dynamic app creation system accessible to authorized users
- Wizard interface for defining:
  - App name and description
  - Custom icon selection
  - Input field configuration
  - Payment model setup (subscription/one-time/per-use)
  - Output type specification
- Newly created apps automatically integrate with the universal workflow system

### Roles & Permissions System
Four-tier access control:

1. **Owner**: Full system control including branding modifications, app creation, and user management
2. **Admin**: Operations management, app configuration, payment oversight (no branding changes)
3. **Support**: Customer service functions, refund processing, help desk access
4. **Viewer**: Read-only dashboard access and statistics viewing
